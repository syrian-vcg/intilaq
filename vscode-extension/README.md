# لغة الانطلاق — إضافة VS Code

إضافة لمحرر VS Code تضيف دعم تلوين الصياغة (Syntax Highlighting) لملفات لغة **الانطلاق**،
وهي لغة بيانات عربية بسيطة لتنظيم وتبادل البيانات بين الأنظمة المختلفة.

## الميزات
- تلوين الكلمات المحجوزة (٢٩ كلمة): `اسم`, `معرف`, `مجموعة`, `حلقة`, `تعريف`, `حاجز`, `نص`, `كتلة`, `كائن`, `قص`, `لصق`, `نقل`, `توريد`, `تصدير`, `تنزيل`, `طباعة`, `جمع`, `قسم`, `ترجمة`, `فقرة`, `خانة`, `صف`, `هنا`, `اشعار`, `بريد`, `تسمية`, `حدث`, `نجح`, `خطاء`
- تلوين النصوص (بين علامتي تنصيص)، والأرقام، والقيم المنطقية (`صحيح`/`خاطئ`)، والقيمة الفارغة (`فراغ`)
- تلوين التعليقات التي تبدأ بـ `#`
- إغلاق تلقائي للأقواس `{ }` وعلامات التنصيص
- يتعرف على ملفات بامتداد `.intq` أو `.intilaq`

## طريقة التثبيت (تطوير محلي)
1. افتح مجلد `vscode-extension` في VS Code.
2. اضغط `F5` لتشغيل نسخة تجريبية من VS Code تحتوي على الإضافة مفعّلة.
3. افتح أي ملف بامتداد `.intq` (يوجد مثال جاهز في `examples/مثال.intq`).

## طريقة التحزيم والنشر
```bash
npm install -g @vscode/vsce
cd vscode-extension
vsce package
```
سينتج عن هذا ملف `.vsix` يمكن تثبيته مباشرة في VS Code من قائمة:
Extensions → ... → Install from VSIX

## بنية قواعد الصياغة (مرجع سريع)
```
كائن {
    مفتاح = قيمة
حاجز
}

مجموعة {
    حلقة { بند = قيمة }
حاجز
}
```
القيم الممكنة: نص بين علامتي تنصيص، رقم، `صحيح`، `خاطئ`، `فراغ`، أو `كائن{...}`/`مجموعة{...}` متداخلة.
